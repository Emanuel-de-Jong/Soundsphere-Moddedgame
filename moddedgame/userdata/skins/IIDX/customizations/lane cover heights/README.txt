1. Go into the 'jsons' folder.
2. Open 'xx playfield.json'. xx = the gamemode you want to change.
3. Search for the comment that says 'Cover Heights'.
4. Change the 'y' value of the cover you want to change.

Note: If you want to change the Lift. Include these steps:
5. Change the 'y' value of the object under the Lift object.
6. Open 'xx.json'. xx = the gamemode you want to change.
7. Search for the comment that says 'Lift Height'.
8. Change every 'y' value under the comment.
